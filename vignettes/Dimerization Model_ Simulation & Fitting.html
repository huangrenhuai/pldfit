1. Dimerization model

1.1. Dimerization model in equilibration

  • On sensor tip:

    • A+B⇄Kd1ABA+B⇄Kd1AB ⇒Response(nm)=Rmax∗[A]Kd1+[A]⇒Response(nm)=Rmax∗[A]Kd1+[A]

  • In solution :

    • A+A⇄Kd2AAA+A⇄Kd2AA
    Kd2=[A]∗[A][AA]=[A]2[AA]⇒[AA]=[A]2Kd2Kd2=[A]∗[A][AA]=[A]2[AA]⇒[AA]=[A]2Kd2 & [At]=[A]+2[AA][At]=[A]+2[AA]
    ⇒[At]=[A]+2∗[A]2Kd2⇒2[A]2+Kd2∗[A]−Kd2∗[At]=0⇒[At]=[A]+2∗[A]2Kd2⇒2[A]2+Kd2∗[A]−Kd2∗[At]=0
    ⇒[A]=−Kd2+Kd22+8Kd2∗[At]√4⇒[A]=−Kd2+Kd22+8Kd2∗[At]4

  • Combine above equations together, assumming Kd1=Kd2=KdKd1=Kd2=Kd:
    • Response(nm)=Rmax∗(−Kd+Kd2+8Kd∗[At]√)3Kd+Kd2+8Kd∗[At]√Response(nm)=Rmax∗(−Kd+Kd2+8Kd∗[At])3Kd+Kd2+8Kd∗[At]

1.2. Kinetic model of dimerization

  • D1D2 dimerization in solution:

    • A+A⇄Kd2AAA+A⇄Kd2AA
    ⇒[A]=−Kd2+Kd22+8Kd2∗[At]√4⇒[A]=−Kd2+Kd22+8Kd2∗[At]4
    Where: [A][A] is the D1D2 monomer contration.

  • On sensor tip:

    • A+B⇄koffkonAB(Kd1)A+B⇄koffkonAB(Kd1)
    d[AB]dt=kon[A][B]−koff[AL]d[AB]dt=kon[A][B]−koff[AL]

    ⇒⇒ dydt=kon[A](Rmax−R)−koff∗Rdydt=kon[A](Rmax−R)−koff∗R ( Called “simple 1:1 model” )

    ⇒⇒ dydt=kon−Kd2+Kd22+8Kd2∗[At]√4(Rmax−R)−koff∗Rdydt=kon−Kd2+Kd22+8Kd2∗[At]4(Rmax−R)−koff∗R
    Where Kd2=koffkonKd2=koffkon, assuming Kd1=Kd2Kd1=Kd2.

2. Simulation & plotting:

# simulate the data use the following parameters
par = list(kon  = 2e2, koff = 1e-2, rmax = 1)
datsim = list()
datsim$concs <- 1e-5 * (2^(0:5));
datsim$xdata = seq(0, 300, length.out = 1501); # time 
datsim$t2   = 150
datsim$ydata = NULL

# simulation
xySimulated <- dfdimer(par = par, dat = datsim, noise = 0.01) 

# plot the simulation
# ySimulated$Time = time; 
xy <-reshape2::melt(data = xySimulated, 
                     id.vars = "Time", 
                     measure.vars = rev(1:6), 
                     variable.name = "Conc")

g <- ggplot()  + xlab("Time (sec)") + ylab("Response (nm)") +
    labs(linetype= 'title') + 
    ylim(-0.025,1) + 
    theme_classic() + 
    theme(legend.position=c(0.9, 0.65),
          legend.text=element_text(size = rel(1)),
          legend.key.size=unit(0.9,"line"));
g <- g + geom_line(data = xy, aes(x = Time, y = value, color = Conc));
print(g)

3. Fitting & plotting:

# init
initPar_test = list(kon =1, koff = 1, rmax = 1) 
dat_test <- datsim
dat_test$t2 <- 149.9 # t2 is the beginning of the diassociation. 
dat_test$lowerBound = list(kon =1e-04, koff=1e-04, rmax = 0.01);
dat_test$upperBound = list(kon =1e04, koff=1e04, rmax = 10);
dat_test$datF       = within(xySimulated, rm("Time")); 

# Fit
fit <- kinfit(par = initPar_test, dat = dat_test, model = "dimer")
