Simple One to One Binding: Simulation

Ren-Huai Huang

2016-02-22

Simple one to one model:

A+L⇄koffkonALA+L⇄koffkonAL

  • A: the protein to be analyzed.
  • L: Ligand immobilized on the pin
  • AL: Complex formed on the pin

Below is the rate equation:

d[AL]dt=kon[A][L]−koff[AL]d[AL]dt=kon[A][L]−koff[AL]

  • other forms:

    • dydt=kon[conc](Rmax−R)−koffydydt=kon[conc](Rmax−R)−koffy

    • dydt=kon[conc]Rmax−(kon[conc]+koff)ydydt=kon[conc]Rmax−(kon[conc]+koff)y

    • dydt=konCRmax−(konC+koff)Rdydt=konCRmax−(konC+koff)R

  • Parameters:

    • Response (nm): R=[AL]=yR=[AL]=y
    • Ligand on the sensor: [L]=Rmax−R[L]=Rmax−R
    • Conc. of the analytes: C=[conc]=[A]C=[conc]=[A]

    • Rmax=[AL]+[L]Rmax=[AL]+[L]

Simulation & Plot

# simulate the data use the following parameters
par = list(kon  = 2e2, koff = 1e-2, rmax = 1)
datsim = list()
datsim$concs <- 1e-5 * (2^(0:5));
datsim$xdata = seq(0, 300, length.out = 1501); # time 
datsim$t2   = 150
datsim$ydata = NULL

# simulation
xySimulated <- dfsimple1to1(par = par, dat = datsim, noise = 0.01) 


# plot the simulation
# ySimulated$Time = time; 
xy <-reshape2::melt(data = xySimulated, 
                     id.vars = "Time", 
                     measure.vars = rev(1:6), 
                     variable.name = "Conc")

g <- ggplot() + theme_classic() + xlab("Time (sec)") + ylab("Response (nm)") +
    labs(linetype= 'title') + ylim(-0.025,1) + 
    theme(legend.position=c(0.9, 0.65),
          legend.text=element_text(size = rel(1)),
          legend.key.size=unit(0.9,"line"));
g <- g + geom_line(data = xy, aes(x = Time, y = value, color = Conc));
print(g)

Fitting

# init
initPar_test = list(kon =1, koff = 1, rmax = 1) 
dat_test <- datsim
dat_test$t2 <- 149.9 # t2 is the beginning of the diassociation. 
dat_test$lowerBound = list(kon =1e-04, koff=1e-04, rmax = 0.01);
dat_test$upperBound = list(kon =1e04, koff=1e04, rmax = 10);
dat_test$datF       = within(xySimulated, rm("Time"));
